## Technology

SodsBC is a post-quantum Byzantine fault-tolerant blockchain consensus protocol built from the ground up with quantum-resistant primitives (secret sharing, permutation-based cryptography, asynchronous verifiable secret sharing, reliable broadcast). The protocol and its security proofs are published in peer-reviewed venues; a Hyperledger Besu implementation is under active development.
